# 📊 EmpTrack Pro

An intelligent **Employee Lifecycle, Probation & Analytics Management System** designed to streamline HR operations using a structured database and web-based interface.

---

## 🚀 Project Overview

EmpTrack Pro is a database-driven system that manages employee records, departments, job roles, and probation tracking. It ensures structured data handling, automation of HR processes, and efficient decision-making through a MySQL relational database and a responsive web interface.

---

## 🧰 Tech Stack

- 🌐 Frontend: HTML, CSS, JavaScript  
- 🧠 Backend: Python (Flask Framework)  
- 🗄️ Database: MySQL (MySQL Workbench)  
- 🔧 Version Control: Git & GitHub  
- 📊 Design Tool: Lucidchart (ER Diagram)  
- 💻 IDE: Visual Studio Code

## 🗃️ Database Design

The system uses a relational database with key entities:

- Employee  
- Department  
- Job Role  
- User  
- Probation Records  

Relationships are carefully designed to ensure data integrity and efficient querying.
